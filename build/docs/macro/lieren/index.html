技能绑定宠物进攻宏​

/petassist 是《魔兽世界》中的一个宠物命令，用来设置宠物的行为模式为 协助模式。

在游戏中，宠物有以下四种主要行为模式：

  • 防御模式（/petdefensive）：宠物会优先攻击攻击你的敌人或你的目标。
  • 被动模式（/petpassive）：宠物会停止攻击，不会主动行动，完全由你手动指挥。
  • 协助模式（/petassist）：宠物会自动协助你攻击当前目标，但不会主动攻击其他目标。
  • 攻击模式（/petaggressive）：宠物主动攻击附近的敌人。
#showtooltip
/cast 技能
/petassist
/petattack
